# Turbo Racer Neon Highway

Turbo Racer Neon Highway is an exciting car racing game developed using Google AI Studio.  
Race on neon-themed highways and compete against AI opponents in fast-paced action.

## Features
- Multiple cars to choose from  
- Smooth and responsive controls  
- Neon-themed highway graphics  
- Challenging AI opponents  

## How to Play
1. Download or clone this repository.  
2. Open the `index.html` file in your browser.  
3. Use the arrow keys to control your car:  
   - ⬆️ Up Arrow → Accelerate  
   - ⬇️ Down Arrow → Brake/Reverse  
   - ⬅️ Left Arrow → Move Left  
   - ➡️ Right Arrow → Move Right  
4. Race against AI cars and try to reach the finish line first!
